Pitch 38 · Home Robot Skill Layer

HomeCore

家用机器人的技能内核：像安装 App 一样安装机器人技能，用 LeRobot 学习，用 Qualcomm 边缘 AI 本地执行。它不是万能管家，而是可演示、可验证、可更新的家庭技能系统。

查看家庭技能 连接 SkillCertKit

01 · Skill Library

家庭机器人不要先承诺“什么都会”，而是先把小技能做可靠。

消费机器人已经从扫地、割草和陪伴开始规模化，但家庭操作仍很难。HomeCore 把家务拆成有边界的技能卡：能做什么、不能做什么、何时暂停、何时请求帮助、如何更新。

Table Reset

收拾轻量杯子、纸巾、零食盒和托盘，不处理刀具、热液体和易碎风险。

Toy Bin

把识别出的软玩具和安全物品放回指定区域。

Entry Check

检查门口、灯光、包裹和家庭设备状态，生成家庭摘要。

Fetch Assist

在已映射房间内递送轻量日用品，低速、低力、需确认。

Pet Zone

识别宠物区域和禁行区，避免把宠物监督当成安全承诺。

Smart Home

在用户授权下联动灯、空气净化器、扫地机器人和提醒设备。

02 · Privacy-First Edge

家庭数据不应该默认离家。

家庭机器人看到的是客厅、老人、孩子、宠物、日程和生活习惯。HomeCore 的默认姿态是本地感知、本地语音、本地地图、本地执行；只有在用户明确授权时，才上传脱敏片段用于训练和远程家人支持。

  • Local-only mode：摄像头、麦克风、地图和家庭事件摘要优先在本体侧处理。
  • Consent panel：家庭成员、照护者和远程支持各自拥有明确权限。
  • Visible sensing：摄像头/麦克风状态、上传状态、最近模型更新和审计日志可见。
  • Data minimization：上传的是经选择的示教片段、失败片段和技能指标，不默认上传全屋视频。

03 · Family Co-Care

帮助家人减少焦虑，不替代照护者。

HomeCore 的养老叙事是居家支持和家庭协同：提醒、日常状态摘要、物品递送、智能家居联动和服务派单。它不做医疗诊断、不承诺急救响应、不宣称替代护理人员。

China 面向成人子女、社区养老站、物业、运营商和智能家居生态，主张“陪伴 + 安全 + 居家养老中枢”。
Overseas 面向家庭照护者、home-care agency、senior living 和 payer pilots，主张 caregiver relief 与 privacy。
Subscription 硬件 + 月费：家庭 app、提醒、技能更新、远程家人摘要、设备联动和可选服务派单。
Consent 远程支持只在老人和家庭授权范围内显示摘要、提醒和协助请求。

04 · Teach To Skill

演示一次，训练成技能；不确定时，机器人会停下来。

HomeCore 的技术亮点不是让机器人乱猜，而是人机协同训练闭环：示教、LeRobot 数据集、云端训练、评估、Qualcomm edge 部署、现场失败回流。

Install

选择 Table Reset、Toy Bin、Entry Check 等家庭技能。

Teach

用户或远程专家示教 10-20 次，生成 LeRobot-compatible 数据。

Train

中国云或海外云训练，记录数据授权、评估和版本。

Deploy

通过 SkillCertKit / SafetyOps 后，签名部署到 Qualcomm 本体。

Correct

机器人遇到未知物品会暂停请求帮助，纠正片段进入下一轮训练。

05 · Home ODD

每个技能都有家庭边界。

HomeCore 只承诺在明确家庭 ODD 内工作：室内、干燥、平整地面、已映射区域、低速、低力、可见急停、用户确认和可回滚技能。家庭机器人会做，也知道什么时候停下来。

不做医疗

不诊断跌倒、中风、痴呆或抑郁，不替代医生、护士或照护者。

不碰危险物

不处理刀具、热液体、炉灶、火源、工具、碎玻璃和高风险清洁剂。

不做监护承诺

不承诺儿童、宠物或老人无人看护安全；只做授权提醒和协助。

不越界运行

不走楼梯、浴室湿地、阳台、车库、户外路线或未验证区域。

06 · Why Qualcomm

家庭机器人真正需要的是本地 AI、低功耗、多模态和长期更新。

HomeCore 让 Qualcomm 成为家用机器人的隐私边缘底座：本地语音、视觉、地图、物体识别、智能家居联动和低速操作都在本体侧完成；云端只负责授权训练、模型优化和技能更新。

Edge RB3/QCS6490 级别可做 MVP，IQ-8275/IQ-9075 可作为多模态和多摄像头升级路线。
Privacy 家庭视频、语音和地图优先本地处理，上传开关和审计记录可见。
Matter 智能家居控制采用 Matter-compatible where supported，不承诺万能联动。
Skills AI Hub、LeRobot、SkillCertKit 和 SafetyOps 形成技能训练、评估、部署和回滚闭环。

HomeCore 的商业价值不是宣布“家务自动化已经完成”，而是把家庭机器人能力做成可安装、可示教、可评估、可本地执行、可持续更新的技能系统。
