HardwareMap

RobotMac Core 不是给开发板套壳，而是一条可销售硬件产品线：Lite、Pro、Industrial 和未来 IQ10 Robotics，软件体验保持统一。

硬件 SKU 打开控制台原型

01 · Product SKUs

用不同 Qualcomm 硬件承担不同价格带和场景。

统一的不是开发板，而是 CloudTwin、SkillDock、EdgeFleet 和部署包体验。硬件按成本、算力、IO、实时控制和传感器规模拆成 SKU。

Core Lite

Radxa Dragon Q6A / QCS6490。面向教育、小车、轻量机械臂和低成本开发者套件。

Core Pro

APLUX Rhino X1 / QCS8550。比赛主线，强调 48 TOPS INT8、边缘 AI、多媒体和标准赛道落地。

Core Industrial

Arduino VENTUNO Q / IQ-8275 + STM32H5。强调 AI compute 和确定性执行的双脑架构。

Core IQ10

未来 production-ready robotics reference design 路线，面向 sensor-heavy AMR、人形和高端服务机器人。

02 · BOM Blocks

真正的一体机，必须把计算之外的麻烦也产品化。

  • Compute：Qualcomm edge AI board、存储、网络、设备身份。
  • Power：电池/外部供电、稳压、保险丝、过流、状态灯。
  • Robot IO：摄像头、CAN-FD、UART、PWM、GPIO、IMU、雷达扩展。
  • Safety：急停、watchdog、本地安全状态机、断云可运行。
  • Thermal：金属外壳、散热器、风扇选项、安装导轨、线缆固定。

03 · Safety Boundary

电源和安全是机器人产品化的分水岭。

评委看到一体机，不应只看到算力盒子，而要看到商业化成熟度：安全供电、本地控制、故障回退、急停、断网可运行、热管理和可维护外壳。

急停 物理 emergency stop 与本地控制回路绑定。
隔离 逻辑电源和电机电源分域，减少调试风险。
回退 模型和技能包可回滚，本体保留稳定版本。
断云 云训练可选，本地执行和安全边界必须保留。
