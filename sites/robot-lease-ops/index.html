00 · Five-Thread Research

RaaS 不是付款方式，而是让机器人变成可续费产能的商业控制面。

五条研究线说明：仓储、制造、cobot 和中国租赁市场已经出现 RaaS；CFO 关心现金流、验收、风险和产能；服务系统决定毛利；中国与海外融资路径不同；Qualcomm 可以通过订阅包绑定 AI Hub、OTA、SafetyOps 和 EdgeFleet。

RaaS Models

Brightpick、Locus、Geek+、AutoStore、Formic、UR leasing 都证明市场接受度。

CFO Logic

买家不是只算人工替代，而是算现金流、可用产能、停机风险和 payback。

Service System

覆盖范围、SLA、维护、备件、RMA、远程支持和客户成功决定合同质量。

Regional Finance

中国做政策 / 银行 / 租赁 / 补贴 ready，海外做 ROI / 税务 / lender ready。

Qualcomm

订阅包让 AI Hub、OTA、FoundriesFactory、SafetyOps 和 EdgeFleet 成为月费价值。

01 · Bundle Builder

把机器人从采购项目，变成可验收、可月付、可扩容的产能包。

RobotLeaseOps 的报价不是“这台机器人多少钱”，而是客户选择场景、班次、吞吐、SLA 和服务等级后，生成 CapEx、lease、RaaS、pay-per-task 四种路径。

Scenario

AMR、lab transfer、machine tending、inspection、palletizing。

ROI

人工、加班、停机、返工、订单波动、维护和内部能力缺口。

Contract

购买、融资、租赁、RaaS、按任务、按小时、季节性扩容。

SLA

Core 97.5%、Pro 98.5%、Critical 99% 生产时段可用率。

Invoice

月费、用量、服务积分、停机扣减、备件和扩展机器人。

02 · Acceptance Before Billing

企业愿意订阅的前提，是机器人先通过验收。

RobotLeaseOps 把收费起点绑定到 PilotContractKit 和 IntegratorForge 的验收证据：不是到货就收费，而是跑过明确的生产 KPI 才进入正式订阅期。

Throughput

每小时 pick、move、cycle、inspection 或 lab run。

Availability

合同生产时段 uptime、计划维护和异常排除规则。

Quality

first-pass yield、误抓、漏检、返工和人工接管率。

Safety

安全事件、near-miss、SafetyOps gate 和 rollback proof。

Service

MTTR、响应窗口、备件到位、远程诊断和现场支持。

Signoff

3-10 个连续班次或 30 天稳定运行后的 operator signoff。

03 · Service Ledger

RaaS 成败不在合同模板，而在每台机器人是否被持续照看。

RobotLeaseOps 维护每台机器人从交付到续约的商业状态：coverage、warranty、lease term、battery、sensor、model version、OTA ring、service ticket、spare parts、telemetry consent 和 residual value。

  • Coverage Ledger：质保、服务等级、免责、易耗件、人工 / 差旅 / 备件范围。
  • SLA Engine：可用率、响应窗口、修复目标、service credit 和 customer-caused fault。
  • Maintenance：小时、循环、报警、电池、减速器、传感器和例检自动生成工单。
  • Remote Cyber：MFA、JIT access、session log、segmentation 和 vendor risk。

04 · Qualcomm Subscription Layer

把 Qualcomm edge 从 BOM 里的芯片，变成月费里的核心证据。

每个 RobotLeaseOps bundle 都绑定 Dragonwing / RB / QCS target、AI Hub validation、EdgeRuntimeBench、OTA ring、SafetyOps evidence 和 EdgeFleet monitoring。这样 Qualcomm 进入的不只是硬件销售，而是持续运营。

Entitlement 模型、runtime、AI Hub profile、QNN artifact 和设备身份绑定。
Telemetry 利用率、热、功耗、battery、sensor、model version 和任务结果。
OTA FoundriesFactory / RobotCoreOS ring、canary、rollback 和 evidence。
Renewal 真实部署数据推动续约、扩容、保险、备件和下一个技能包。

RobotLeaseOps 把“买机器人”改写成“订阅一段可验证、可维护、可扩张的生产能力”。
