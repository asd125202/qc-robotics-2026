01 · Deployment System

不要做集成商黄页，要做可认证交付系统。

IntegratorForge 的产品不是“找一个人来装机器人”，而是把伙伴资质、项目模板、硬件兼容、集成证据、验收门禁和服务 SLA 串成一个交付操作系统。

Audit

workflow survey、ROI model、safety scan、integration map 和现场约束。

Match

按地区、行业、应用、认证等级、硬件经验和 SLA 匹配集成商。

Recipe

BOM、sensor、EOAT、ROS 2 graph、QNN asset、connector 和 safety gate。

Accept

FAT / SAT / SIT、照片视频证据、签名、punch list 和 retest history。

Serve

installed base、spares、SLA、remote triage、CMMS export 和变更管理。

02 · Reusable Project Library

每一次交付，都应该沉淀成下一次更快成交的 recipe。

成熟生态的共同点是：认证组件、应用模板、培训、伙伴定位、销售协同、技术支持和质量分层。IntegratorForge 把这些做成 RobotMac / Qualcomm-first 的交付库。

Warehouse AMR

地图、路线、充电、WMS/WES 连接、R15.08 对齐和现场 hypercare。

Machine Tending

夹具、EOAT、PLC、节拍、风险评估、培训和操作员交接。

Palletizing

负载、码垛模式、安全围栏、视觉校准、FAT/SAT 和 OEE 指标。

Lab Cell

LIMS/ELN、样品 lineage、协议变更、校准、审计和合规证据。

Migration Kit

Jetson / RK / RPi / Arduino 项目迁移到 Dragonwing edge target。

Training

online seat、hands-on lab、train-the-trainer、exam 和 annual renewal。

03 · Handover Binder

企业验收要的不是宣传页，而是可审计的交付证据包。

IntegratorForge 自动生成项目 handover binder：site survey、risk assessment、FAT/SAT/SIT、commissioning、operator training、maintenance plan、open punch list、change history 和 Qualcomm edge runtime evidence。

  • Risk Pack：standards-aware assessment、risk-reduction hierarchy 和 validation evidence。
  • FAT/SAT Runner：测试步骤、见证签名、照片视频、punch list 和复测历史。
  • MOC Module：机器人程序、EOAT、payload、map、firmware、PLC、HMI 变更都触发复验。
  • Field Service：SLA、备件、远程诊断、现场服务和 CMMS / ITSM 导出。
