Pitch 65 · Surface Release Packet Layer

表面工程最贵的不是打磨和喷涂，而是没人能证明这一平方米可以放行。

SurfaceLoop 是船厂、重钢、风电、汽车和航空表面工程的放行证据层：把前处理、环境、膜厚/粗糙度、缺陷、机器人/人工返工、检验员签核和付款/质保证据变成一个可追溯 release packet。

看痛点 看解决方案

01 · Problem

表面工程的失败，通常不是因为没人干活，而是证据在交付前断掉。

船体、储罐、桥梁、风电塔筒、汽车车身和航空蒙皮的表面工作跨越前处理、涂装、复检、返修和放行。照片在手机里，膜厚在仪器里，环境数据在纸单里，返工口径在群聊里，最后真正决定付款和质保的 release packet 仍靠人手拼。

业主 / 总包

关心的是停航、停线、干坞延期、质保争议和付款依据，不是又多一个缺陷截图。

船厂 / 涂装主管

多班组、多区域、多设备同时施工，真正拖慢交付的是返工票、复检排队和资料退回。

承包商 / QC

前处理或涂装问题发现太晚，返工吃掉利润；责任边界没有证据，争议只能靠人扯。

第三方检验员

需要把照片、DFT、profile、温湿度、批次、返工和签名整理成可审计文档。

02 · Why Now

2026 年的窗口打开了：腐蚀成本高、熟练工少、机器人开始进表面工序，监管要证据。

过去表面工程靠老师傅、抽检和纸质表单凑合。现在大型资产交付节奏更快，船厂和车厂开始部署表面机器人，AI+制造政策要求在线检测和质量追溯，客户也更难接受“我看过了”的口头放行。

$2.5T AMPP/NACE 常用口径：全球腐蚀年度成本约 2.5 万亿美元，成熟控制措施可显著节约。
84.9% 新华社报道，2026 年一季度中国新接船舶订单全球份额达到 84.9%，船厂质量和交付压力继续上升。
120GW 国家能源局数据：2025 年中国新增风电装机约 120GW，塔筒、叶片和海上维护带来重复表面检验需求。
773k 国家统计局口径：2025 年中国工业机器人产量 773,074 台，机器人执行层正在进入更多非结构化工序。

03 · Core Insight

涂料公司卖材料，机器人公司卖动作；客户真正买的是可付款、可质保、可审计的放行决定。

SurfaceLoop 的核心判断：表面工程最有价值的数据，不是孤立的锈蚀图片，而是从表面初始状态到最终放行结果的完整因果链。谁做了、怎么做的、测到了什么、为什么返工、谁签核、能否付款，这才是客户愿意付费的系统。

Release Packet First

先围绕放行包设计产品，再决定用相机、机器人、传感器还是人工补拍来补证据。

Inspector-in-the-Loop

AI 标记风险，检验员批准、退回或要求补拍；系统不替代持证检验和业主签核。

Rework Becomes Data

低置信度、返工、复检、人工接管和改判都变成 LeRobot/HIL 训练数据。

04 · Solution

SurfaceLoop 是设备中立的 Surface Release Packet Layer。

它不和 GrayMatter、Blastman、VertiDrive、AMBPR、KUKA、FANUC、Gecko 或 CMMS/MES 争同一个位置。它坐在机器人、巡检、涂装设备、手机和仪器之上，把现场动作变成可关闭的表面工作包。

  • Spec：导入资产、区域、涂层体系、前处理标准、hold point、责任班组和放行条件。
  • Capture：采集照片、视频、DFT/profile、温湿度、批次、机器人轨迹、工具状态和检验员批注。
  • Flag：端侧识别锈蚀、残留旧涂层、漏打磨、针孔、橘皮、膜厚风险和低置信度区域。
  • Route：把问题派给人工、机器人、承包商或第三方复检，而不是停在一张缺陷热力图。
  • Release：生成 before/after、测量、模型版本、签名、返工历史和 owner-ready release packet。

05 · Product Workflow

从“这块表面做完了”到“这块表面可以放行”，系统给出完整证据。

第一版切口不是全自动喷涂，而是 shipyard / heavy steel 的 coating release work package。一个工作包跑通后，再扩展到储罐、桥梁、风电、汽车漆面和航空 MRO。

1. 建包

把区域、标准、材料、天气窗口、检验点和付款/质保条件转成数字工作包。

2. 采证

边缘节点连接相机、膜厚仪、粗糙度仪、环境传感器、机器人和手机补拍。

3. 开票

AI 只负责提示风险和证据缺口，自动生成 NCR / rework ticket / needs-review 区域。

4. 闭环

返工动作、低力接触、复检照片、测量值和人工接管都绑定同一 surface zone。

5. 放行

检验员签核，业主复核，系统输出 release packet 并回写 MES/CMMS/EAM/Ship OS。

06 · Market Wedge

先从“船厂和重钢表面放行”切入，因为一次返工或一天延期就能证明 ROI。

这是一个足够窄但足够贵的切口：大面积、多班组、强检验、弱网络、数据敏感、返工昂贵、争议频繁。SurfaceLoop 先赢下一个放行工作包，再让业主把它写进更多项目的 QA 交付格式。

船厂 / 海工

分段、压载舱、甲板、舱室和干坞维修需要前处理、涂装、复检和业主签核。

储罐 / 管廊

停产窗口昂贵，防腐层、变形、腐蚀、补漆和交接资料都需要可追溯。

桥梁 / 钢构

中国桥梁和钢构资产规模大，新维护标准和业主审计推高数字化记录价值。

风电塔筒 / 叶片

吊装、停机和海上维修成本高，表面缺陷必须定位、复检、归档和追踪。

汽车漆面

NEV 快速换型、VOC 升级和外观质量压力推动漆面检测与返修闭环。

航空 MRO

scuff sanding、涂层修补和停场时间要求清晰的前后证据和人工签核。

07 · Business Model

不是卖表单软件，而是卖 constraint-day insurance 和 defensible release packet。

客户买单的理由很直接：放行包准备更快、资料退回更少、返工更早发现、付款更快、争议更少。定价从一个项目工作包开始，避免一上来卖大而全平台。

90 天试点

海外 7.5k-15k 美元/项目工作包；中国 5-10 万元。覆盖模板、5-10 用户、现场 onboarding 和 ROI 报告。

年度项目订阅

18k-36k 美元/站点或项目团队/年，外加 75-150 美元/现场用户/月；业主/检验员查看免费。

企业业主版

50k-150k 美元/年，覆盖多团队、SSO、审计、数据留存、owner reporting 和系统集成。

边缘证据节点

按节点租赁或一次性硬件加软件维护，负责多相机接入、弱网缓存、模型 profile 和证据同步。

08 · Go-To-Market

90 天只证明四件事：更快编包、更少退回、更早发现返工、更快放行。

采购路径从承包商或船厂 QA/生产预算进入，拿一个真实可计量的涂装/防腐工作包做试点。成功后推动业主、总包或船级/项目控制把 SurfaceLoop release packet 写进交付要求。

Packet Time

release packet 编制时间下降 30%-50%，报告从 inspection 到 review 控制在 24 小时内。

Return Rate

资料退回/缺字段率下降 50%，关键 hold point 和环境/膜厚/profile/批次记录完整率接近 100%。

Rework Catch

漏打磨、残留旧涂层、膜厚风险等问题在下一道涂层前被发现并关闭。

Owner Pull

当业主发现它能减少争议和加快付款，下一步就是要求承包商统一交这种包。

09 · Competition

这个市场已经有人做机器人、巡检和 AOI；SurfaceLoop 必须占据“工作包闭环”位置。

GrayMatter、Blastman、VertiDrive、AMBPR、KUKA、Durr、FANUC 和 FerRobotics 强在执行；Gecko、Square Robot、Apellix、Voliro 强在巡检；Cognex、Keyence、Hikrobot、MVTec 强在机器视觉；SAP、Maximo、MES/CMMS 强在记录。缺口是跨设备、跨班组、跨检验点的放行证据层。

不是机器人本体

机器人做动作；SurfaceLoop 证明动作发生在正确区域、正确条件下，并被正确复检。

不是视觉缺陷库

检测缺陷只是开始，付费的是缺陷如何被派工、返修、复检、签核和归档。

不是 CMMS/MES

现有系统记录工单状态；SurfaceLoop 记录表面证据、模型建议和检验员决策。

不是替代检验员

产品把检验员放进闭环：AI 建议、人工签核、低置信度回流训练。

10 · Moat

护城河不是一套模型，而是表面状态到放行结果的证据图谱。

SurfaceLoop 的数据比普通图片更贵：它带资产、区域、标准、材料、环境、工具、力控、测量、返工、检验员处置、业主签核和最终放行结果。模型越跑，工作包越标准，跨设备迁移越容易。

Surface Evidence Graph

把 before/after、DFT/profile、环境、缺陷、返工动作、复检和签核连成可查询图谱。

Workflow Lock-in

业主、承包商、检验员、项目控制和质保团队共用同一个 release packet 格式。

Equipment Neutrality

可接机器人、爬壁车、无人机、手机、膜厚仪、粗糙度仪和既有 MES/CMMS。

Edge Deployment Know-how

Qualcomm 模型 profile、离线缓存、弱网同步、回滚和数据不出厂变成部署资产。

11 · Architecture

云端训练，现场推理，控制和签核留在确定性边界内。

比赛 demo 使用安全桌面 coupon，不做真实喷涂、溶剂、金属粉尘或高能打磨。DFT/profile 字段默认模拟，只有接入校准仪器时才作为真实测量。机器人控制由 ROS 2 和安全 supervisor 执行，AI 不直接进入安全闭环。

Field Capture

双 RGB 相机、可选 depth/thermal、load cell 或 F/T、二维码、环境和工位状态。

Edge AI

缺陷分割、coverage/risk flag、contact-state classifier、before/after quality score。

Control Boundary

ROS 2、速度/力上限、geofence、工具联锁、E-stop、HIL 暂停和人工恢复。

Dataset

LeRobotDataset v3 保存多相机视频、state/action、force、intervention、label 和模型版本。

Release API

输出 PDF/JSON release packet，并回写 MES、CMMS、EAM、QMS、Ship OS 和文控系统。

12 · Competition Demo

3 分钟证明：表面缺陷被发现、返工、复检、签核，并回流训练。

演示用水性涂层 coupon、记号笔划痕、胶带 ridge 或 clearcoat dot 模拟缺陷；用低力 felt/foam pad 和安全夹具展示动作闭环。重点是证据链，不是危险工艺。

0:00 Scan

边缘相机识别模拟 dust nib、划痕、薄涂区域或漏打磨点，打开 surface NCR。

0:45 Probe

低力探针或 load cell 记录 contact trace；mock DFT/profile 被明确标注为模拟字段。

1:20 Correct

故意触发 mask drift 或漏检，人工 HIL 暂停、接管、修正并恢复策略。

2:05 Edge

小模型在 Qualcomm edge 上离线跑 pass/fail/needs-review，云端不参与控制。

2:45 Release

系统生成 release packet，并把干预片段写入 LeRobotDataset 作为下一轮训练数据。

13 · Why Qualcomm

Qualcomm 可以把“机器人硬件 + 本地 AI + 现场证据”做成工业表面工程参考应用。

表面工程发生在弱网、粉尘、噪声和数据敏感的现场。多相机、低功耗边缘推理、弱网缓存、AI Hub/QNN profile、ROS 2 机器人运行时和中国/海外双部署，正好是 Qualcomm 能形成生态拉力的位置。

Edge Evidence

原始现场图像不默认出厂；边缘先做筛查、压缩、脱敏和本地放行证据生成。

AI Hub / QNN

缺陷分割、质量评分和 evidence classifier 可通过 AI Hub compile/profile 后下发到目标 SoC。

Robotics Runtime

RB3 Gen 2 / QCS6490 级边缘节点可作为相机、传感器、ROS 2 和证据包的现场枢纽。

China + Global
