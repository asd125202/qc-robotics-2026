# 第一批产品概念组合

## 概念 1：RobotMac Core

定位：机器人行业的“Mac mini + 电源管理 + IO 扩展 + AI 运行时”。

用户不再从开发板、电源线、散热、驱动、传感器同步开始，而是购买一个可装到机器人里的核心模块。

核心卖点：

- 计算、电源、IO、安全、散热一体化。
- 本地闭环控制，不依赖公网。
- 面向 Qualcomm Dragonwing QCS8550 / QCS6490 / IQ-8275 做迁移路线。
- 默认对接 LeRobot 数据采集和部署包。

商业化：

- 硬件毛利 + 扩展模块 + 企业支持。
- 更像 “robot computer” 而不是开发板。

## 概念 2：LeRobot CloudTwin

定位：机器人训练的云端双生系统。

机器人在本体采集真实数据，云端训练策略，再把模型发布回本体。每一次失败和人类接管都会变成下一轮训练数据。

核心卖点：

- 数据集版本化。
- 中国/海外双云适配。
- ACT 保底，SmolVLA/X-VLA 前瞻。
- 云端训练，本体推理，安全闭环留在本地。

商业化：

- SaaS 订阅 + GPU 计费抽成 + 企业私有化部署。

## 概念 3：SkillDock

定位：机器人技能商店。

开发者训练技能，用户像安装 App 一样安装到不同机器人形态上。Qualcomm 设备成为可验证、可运行、可分发的边缘目标。

核心卖点：

- 技能包包含模型、依赖、输入输出契约、安全边界和评估报告。
- 机器人不再只卖硬件，也卖可升级能力。
- Qualcomm 可成为“机器人技能认证硬件平台”。

商业化：

- 技能市场抽成。
- 企业私有技能库。
- 认证设备和认证技能。

## 概念 4：Embodied Kit

定位：可采购、可交付、可复现的行业机器人套件。

面向工厂、实验室、教育和服务公司，提供打包好的硬件箱、机器人本体、云训练账号、示例技能和部署文档。

核心卖点：

- 从比赛 demo 到行业试点的最短路径。
- 降低企业采购风险：不是散件，而是套件。
- 可围绕巡检、分拣、质检、教学、服务等行业复制。

商业化：

- 套件销售 + 行业模板 + 现场集成 + 维护订阅。

## 首轮取舍

比赛初赛建议主叙事采用 `RobotMac Core + LeRobot CloudTwin`，因为它最能同时解释商业价值、技术完整性、Qualcomm 必要性和可落地性。

`SkillDock` 和 `Embodied Kit` 更适合作为商业扩展章节，证明这不是单一项目，而是可以长成生态的产品路线。
