# Milestones

## Preliminary Round

### Before 2026-07-20

- Register for the competition.
- Submit the project proposal.
- Freeze the initial project name, problem statement, and demo scenario.
- Publish a simple project page or README-based demo.

## Shortlist / Board Preparation

### Before 2026-07-31

- Confirm shortlisted status.
- Track development board shipment.
- Prepare board bring-up checklist.
- Prepare fallback simulation or laptop demo path.

## Final-Round Development

### 2026-08 to 2026-11-01

- Bring up the development board.
- Port model inference to the target runtime.
- Integrate sensors and robot control loop.
- Record reproducible demo runs.
- Prepare online final submission package.

## Awards

### End of 2026

- Prepare conference presentation assets if selected.
- Archive source, models, demo videos, and deployment notes.

## Weekly Working Plan

- Week 1: project proposal, architecture, demo narrative.
- Week 2: minimal prototype and web demo page.
- Week 3: perception/control prototype.
- Week 4: hardware integration readiness.
- After board arrival: deployment, optimization, validation, video recording.
