# Submission Checklist

## Before 2026-07-20

- [ ] Competition registration complete.
- [ ] Project name selected.
- [ ] Proposal text complete.
- [ ] Architecture diagram or explanation complete.
- [ ] Demo scenario defined.
- [ ] Repository public or accessible as required.
- [ ] Project page deployed if useful.

## Before Final Submission

- [ ] Hardware target documented.
- [ ] Setup instructions tested from a clean environment.
- [ ] Demo video recorded.
- [ ] Model files and licenses documented.
- [ ] Performance metrics recorded.
- [ ] Known limitations documented.
- [ ] Submission package reviewed.

## Security / Hygiene

- [ ] No API keys or private credentials committed.
- [ ] No VPN client profiles committed.
- [ ] No large model weights committed unless explicitly intended.
- [ ] Any third-party code or model license is documented.
