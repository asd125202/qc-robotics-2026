## 最该强调的三句话

1. 这不是一个单次机器人 demo，而是把机器人商业化的低层复杂性压进一个可购买、可训练、可升级的平台。
2. Qualcomm 的价值不只是 TOPS，而是让机器人在本体上低延迟、低功耗、可连接、可迁移、可量产地运行。
3. LeRobot + 双云训练 + Dragonwing edge target 形成数据飞轮，让每次失败都能成为下一轮更好的技能。

## 缺口

- 需要一个更具体的三分钟演示 storyboard。
- 需要一个中文项目书正文版本。
- 需要一个最小可运行 demo：即使没有开发板，也能模拟数据集、训练 job、模型版本和设备指标。
- 需要决定初赛主叙事：建议 `RobotMac Core + LeRobot CloudTwin`，其余作为商业扩展。
