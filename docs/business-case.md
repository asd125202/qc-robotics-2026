# Business Case

更新时间：2026-07-05。

## Core Argument

RobotMac Core 的商业价值不是一次性卖硬件，而是建立一个机器人产品底座：

1. 用硬件套件降低进入门槛。
2. 用云训练订阅让机器人持续变聪明。
3. 用技能市场让能力可复用、可交易、可升级。
4. 用 EdgeFleet 运维把企业客户留在长期服务里。
5. 用 EduForge 教育套件建立未来开发者入口。

## Why Customers Pay

客户不是为“开发板 + 外壳”付费，而是为降低集成风险付费。

| Customer | Current pain | Product offer |
| --- | --- | --- |
| 高校 / 训练营 | 课程时间被硬件调试和环境配置消耗 | EduForge kit + 中文课程 + LeRobot training template |
| 机器人创业团队 | 从电源、驱动、云训练开始太慢 | RobotMac Core Pro + CloudTwin + Prototype dashboard |
| 系统集成商 | 每个项目重复做底层集成 | Embodied Kit + industry templates + EdgeFleet |
| 企业创新部门 | 试点失败风险高、维护不可控 | Procurement-ready kit + fleet operations + service contract |
| 开发者生态 | 技能无法复用和分发 | SkillDock package + Qualcomm-certified targets |

## Revenue Layers

### Layer 1: Hardware Margin

- Core Lite: education and developer entry.
- Core Pro: competition and service robot prototype default.
- Core Industrial: industrial customers and deterministic control.
- Sensors, power modules, IO carriers, and robot kits.

### Layer 2: Cloud Training Subscription

- Team seats.
- GPU-hour budget packages.
- Dataset storage.
- Evaluation reports.
- Deployment package registry.

### Layer 3: Skill Marketplace

- Skill package sales.
- Revenue share with developers.
- Certification fees for Qualcomm targets.
- Private enterprise skill libraries.

### Layer 4: Fleet Operations

- Device monitoring.
- Logs and failure replay.
- Model/skill version control.
- OTA deployment management.
- Support and maintenance contracts.

### Layer 5: Education and Ecosystem

- University kits.
- Training camps.
- Competition templates.
- Course licensing.
- Developer certification.

## Initial Packaging

### Starter

For education and developers.

- Core Lite or simulated cloud dashboard.
- One robot form factor.
- LeRobot data template.
- Limited cloud training credits.

### Pro

For competition and startup prototype.

- Core Pro / QCS8550 path.
- Desktop manipulator or mobile base.
- CloudTwin training loop.
- Prototype dashboard and deployment manifest.

### Enterprise Pilot

For system integrators and companies.

- Embodied Kit.
- EdgeFleet operations.
- Private dataset and skill package.
- Support contract.

## Competition Framing

The proposal should not claim immediate revenue. It should show a credible path:

1. Initial entry through competition demo and developer community.
2. Education kits create repeatable usage.
3. Startup/system-integrator kits create early commercial trials.
4. Enterprise pilots create recurring fleet/skill revenue.
5. Qualcomm benefits because every layer increases Dragonwing edge target adoption.
